LOGO FILES
==========
Put sponsor logo PNGs in this folder. In the Google Sheet, the "Logo" column
holds JUST the filename (e.g. northside-co.png) — not a full path.

Export each logo as a TRANSPARENT PNG at one of these master sizes:
  - Landscape (wide):  1200 x 480 px   -> name shows BELOW the logo
  - Square / upright:   800 x 800 px    -> name shows to the RIGHT of the logo
Leave ~10% clear padding inside the artwork. The page scales them per tier,
so you only export each logo once. Orientation is auto-detected — no extra column.

No logo yet? Leave the Logo cell blank and the sponsor shows as a clean text card.
